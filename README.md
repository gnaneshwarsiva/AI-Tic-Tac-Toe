## 📖 Overview

AI Tic-Tac-Toe is a desktop application developed using **Python** and **Tkinter**. The project demonstrates artificial intelligence concepts through multiple AI difficulty levels, strategic gameplay, and a modular software architecture.

The application allows players to compete against another player or challenge an AI opponent with varying levels of intelligence.

---

## ✨ Features

- 🎮 Human vs Human Mode
- 🤖 Human vs AI Mode
- 🟢 Easy AI (Random Strategy)
- 🟡 Medium AI (Heuristic Strategy)
- 🔴 Hard AI (Minimax Algorithm)
- 💡 Smart Hint System
- ↩ Undo & Redo Moves
- 📖 AI Move Explanations
- 🏆 Automatic Winner Detection
- 🎨 Interactive Tkinter GUI
- ⚡ Fast and Responsive Gameplay
